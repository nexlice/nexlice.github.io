# READ ME
백엔드 개발에서 중요한 데이터베이스를 학습한다.  
SQL문법, 스키마 디자인, Node.js에서 데이터베이스를 사용하는 방법을 알아보자.  
또한 NoSQL과 SQL의 차이점과 더불어, 데이터베이스에서 중요하게 여기는 원칙인 ACID에 대해 다룬다.
