# Read Me
`React`는 웹, 데스크탑, 모바일 등 다양한 플랫폼에서 활용할 수 있는 프론트엔드 라이브러리이다.  
이 문서와 하위 문서에서는 리액트의 개념과 활용 방법에 대해서 다룬다.  

## Prerequisite
- HTML/CSS 기초
- JavaScript 기초
- 함수형 프로그래밍과 고차함수
- 배열 내장 메서드 기초
- ES6 문법에 대한 이해
