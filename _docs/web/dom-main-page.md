# Introduction
`DOM`은 프로그래머 관점에서 바라 본 `HTML`이다.  
`DOM`을 이해하고 조작할 수 있다면, `HTML`을 단순한 문서에서 웹 앱으로 업그레이드 할 수 있다.  
`DOM`은 브라우저 환경에서 `JavaScript`를 이용해 `HTML`을 조작할 수 있다.  
`HTML`문서에 이미 작성되어있는 엘리먼트에 접근하거나, 새로운 엘리먼트를 생성 또는 삭제가능하다.
